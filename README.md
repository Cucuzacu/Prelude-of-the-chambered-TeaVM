# RubyDung-TeaVM

* **You need Java 17 to compile, otherwise it won't work**

Play it here -> https://cucuzacu.github.io/Games/POC

## Building(on Windows)

git clone https://github.com/Cucuzacu/Prelude-of-the-Chambered-TeaVM



cd Prelude-of-the-Chambered-TeaVM



setjavapath



gradlew generateJavaScript





Now, go into the out folder and host it as a web server!


Note: Disable your adblocker. I don't know why but it stops the game from running.
